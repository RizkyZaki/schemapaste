## [0.1.0] - 2026-04-07

### Added

- Initial monorepo setup for extension, webview, and core packages.
- SQL-to-ERD flow with interactive visualization in VS Code webview.
- Sidebar integration through Activity Bar view.
- Export diagram as SVG and PNG.
- Save and load schema snapshots as JSON.
- Migration export scaffolds for Laravel, Prisma, Knex, Sequelize, and TypeORM.
- Core test coverage for parser and migration export generator.

### Changed

- Improved package entry resolution for @schemapaste/core to support webview bundling.
- Upgraded README documentation for marketplace and GitHub onboarding.
